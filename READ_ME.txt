How to run the app:
Since it is a React App you run it by typing "npm start" on your IDE terminal.

How to use the app:
When the app in ran you'll be presented with a button to start.
When that button is pressed you'll be sent to a page with the characters table listed, and a text field to search for a character name.
When you click on the name of any character you'll be sent to a page detailing the character.
You can then return to the table page by pressing the "Return to Character List" hyperlink.
